Support

Bugs, questions, and feature suggestions are all welcome.

Email

Write to support@getwideeye.com. Replies usually come within a couple of days.

Bug reports

For bug reports, including the following helps a lot:

  • macOS version (Apple menu → About This Mac)
  • WideEye version (WideEye menu → About WideEye)
  • What you did, what you expected, what happened instead
  • Whether it's reproducible, and if so the steps

Feature requests

WideEye is intentionally narrow — it's a viewer, not an editor or organizer. Requests that fit the wedge (ultrawide, multi-pane, fast, keyboard-driven) are most likely to land. Requests for editing, tagging, cloud sync, or AI features will probably be politely declined; that's the boundary that keeps the app sharp.
